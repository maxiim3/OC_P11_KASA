## Maxime Tamburrini

### Open Classrooms - Formation React Projet 11

---

You need Sass and TypeScript to run this app

1. Clone Repo :
    ```bash
     git clone git@github.com:maxiim3/OC_P11_KASA.git
    ```
2. Install dependencies :
    ```bash
     npm i
    ```
3. Launch Dev Server :
    ```bash
    npm run dev
    ```
